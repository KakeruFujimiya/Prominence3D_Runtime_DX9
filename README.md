# Prominence3D_Runtime_DX9
Prominence3Dで作ったエフェクトをDirectX9で描画するためのランタイムです。<br><br>
VisualStudio2013用です。<br>
他のバージョンをお使いの場合はブランチを切り替えてください。。<br>
