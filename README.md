# Prominence3D_Runtime_DX9
Prominence3Dで作ったエフェクトをDirectX9で描画するためのランタイムです。<br>
現在VisualStudio2013のみ対応となっています。<br>
他のバージョンへの対応はもうしばらくお待ちください。<br>
